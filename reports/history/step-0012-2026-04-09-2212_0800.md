## What was done
Created 5 Astro components in `src/components/`:

| Component | File | Replaces |
|-----------|------|----------|
| Dict.astro | Dictionary lookup | `{{< dict email from >}}` |
| Person.astro | Person name/link lookup | `{{< mydict2 person name >}}` |
| Original.astro | Document link with prefix | `{{< original en email file >}}` |
| ImgProc.astro | Image in `<figure>` | `{{< imgproc2 file Fit "..." >}}` |
| Signature.astro | LinkedIn + email block | `{{< signature_shortcode_1_ >}}` |

All read locale from `Astro.locals.locale`. Dict and Person use `set:html` to render markdown-to-HTML output.
